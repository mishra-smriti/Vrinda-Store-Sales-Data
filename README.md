# 📊 Vrinda Store Sales Analysis Dashboard 2022

An interactive Excel dashboard project built to analyze sales performance, customer behavior, order trends, and channel-wise contribution for Vrinda Store in 2022.

## Skills Demonstrated

* Data Cleaning
* Data Analysis
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design
* Business Insights Generation

## Dashboard Highlights

* Total Sales Analysis
* Monthly Sales Trends
* Sales by State
* Sales by Gender
* Sales by Age Group
* Sales by Channel
* Order Status Analysis
* Product Category Performance

## Key Insights

* Women customers contributed the majority of sales.
* Amazon was the highest-performing sales channel.
* Maharashtra generated the highest revenue.
* Adult customers (30–49 years) accounted for most purchases.
* Sets and Kurtas were the top-selling categories.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting
* Slicers

## Project Outcome

The dashboard converts raw sales data into actionable business insights, helping stakeholders identify growth opportunities and make data-driven decisions.
